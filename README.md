# Summit Comfort HVAC — Demo Site

A static demo website for a mock HVAC client, built as the first deploy for Web Builder Co.

## Stack
- Single static `index.html` (HTML/CSS/JS, no dependencies, no build step)
- Hosted on Cloudflare Pages
- Domain registered in client's name (when a real client)

## Deploy
Cloudflare Pages serves `index.html` at the root automatically. No build command, no output directory needed — it's pure static.

## Notes
- This is a demo/sales asset, not a live client site.
- Replace content, photos, and colors per client using this as the starting template.
- Any template improvements should be committed back so both partners get them.
